UAVs course Project
Design and Implementation of a GNC software on a Parrot quadcopter
In order to illustrate the theoretical aspects of the Mission and Path Planning course, we had to implement a python version of the RRT algorithm on a 6 DOF robotic arm. The goal of this project was to generate a trajectory so that the robotic arm grasps the ball in the box and, then puts the ball down on the ground at a specified location.
